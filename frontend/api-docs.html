üöÄ Introduction

Welcome to the Email Spam Detector API documentation. This API uses advanced machine learning algorithms to classify emails as spam or legitimate (ham) with high accuracy.

Base URL

http://localhost:8000/api

Features

  • Single email classification
  • Batch processing (up to 100 emails)
  • Confidence scoring
  • Spam indicator analysis
  • Risk level assessment
  • Model comparison (4 AI models)
  • Pattern detection (URLs, emails, phone numbers)

üîê Authentication

Currently, this API does not require authentication. It's designed for local development and demonstration purposes.

Note: For production deployment, implement proper authentication (API keys, OAuth, JWT, etc.)

üì° API Endpoints

GET /api/health/

Check if the API is running and models are loaded.

Response (200 OK)

JSON Response 
{
  "status": "healthy",
  "message": "Spam detection API is running",
  "models_loaded": true
}

POST /api/predict/

Classify a single email as spam or ham.

Request Body

Field Type Required Description
email string Required The email text content to analyze

Example Request

cURL 
curl -X POST http://localhost:8000/api/predict/ \
  -H "Content-Type: application/json" \
  -d '{
    "email": "Congratulations! You won $1,000,000! Click here now!"
  }'
JavaScript (Fetch) 
fetch('http://localhost:8000/api/predict/', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    email: 'Congratulations! You won $1,000,000!'
  })
})
.then(response => response.json())
.then(data => console.log(data));

POST /api/predict-batch/

Classify multiple emails at once (maximum 100 emails per request).

Request Body

Field Type Required Description
emails array[string] Required Array of email texts (max 100)

Example Request

JSON Request 
{
  "emails": [
    "Meeting at 3pm tomorrow",
    "WIN FREE MONEY NOW!!!",
    "Your order has shipped"
  ]
}

Response (200 OK)

JSON Response 
{
  "status": "success",
  "results": [
    {"email": "Meeting at 3pm...", "prediction": "ham", "confidence": 0.92},
    {"email": "WIN FREE MONEY...", "prediction": "spam", "confidence": 0.98},
    {"email": "Your order has...", "prediction": "ham", "confidence": 0.89}
  ],
  "summary": {
    "total": 3,
    "processed": 3,
    "failed": 0,
    "spam_count": 1,
    "ham_count": 2,
    "avg_confidence": 0.93
  }
}
